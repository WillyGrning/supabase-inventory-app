# Inventory Management System

A simple full-stack inventory management application built with React, Express.js, and Supabase.

This project focuses on authentication, session handling, OTP verification, and basic CRUD operations using a straightforward architecture.

---

## Features

- Email & password registration
- Email OTP verification
- Email & password login
- Google OAuth login
- Session-based authentication
- Category management (CRUD)
- Secure API access using Bearer tokens

---

## Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: Supabase (PostgreSQL)
- Authentication: Email, Google OAuth, OTP
